關卡一

網際網路是什麼？

一張連接全世界電腦的巨大蜘蛛網

嘿，探險家！我們每天都在「上網」，但「網際網路」(Internet) 到底是什麼？

想像一下，有一張巨大無比、看不見的蜘蛛網，把全世界的電腦、手機、伺服器（就是存放網站資料的超級電腦）全部都連接在一起。這張大網，就是網際網路！

透過這張網，我們可以：

  • 傳送訊息：就像蜘蛛在網上傳遞震動，我們用 LINE、Email 傳訊息。
  • 獲取資訊：瀏覽 Google、YouTube 上的網站，就像從網上取得掛著的「知識果實」。
  • 分享生活：在 IG、FB 上發文，就是把你自己的故事掛到網上和大家分享。

這張大網，讓資訊可以自由地在全球流動，也徹底改變了我們的生活！

關卡二

深入網路的核心機制

探索數據傳輸的秘密通道

訊息是怎麼在這張大網上跑的？為什麼我在台灣能看到美國的網站？這背後有許多神奇的規則和技術，就像交通系統一樣，我們來一一探索！

訊號傳遞的媒介

數據的傳輸需要媒介，就像聲音需要空氣一樣。常見的有：

  • 雙絞線/電纜：你家裡連到牆壁上的網路線，透過電流傳遞訊號。
  • 光纖：海底電纜就是用這個！用「光」來傳輸，速度超級快！
  • Wi-Fi/手機網路：看不到的無線電波，讓我們可以隨時隨地上網。

IP 位址和 DNS

IP 位址 就像是每台電腦在網路上的「地址門牌」，例如 114.32.227.58。但數字太難記了！

所以我們有了 DNS (網域名稱系統)，它就像是網路世界的「電話簿」，可以把好記的網址 (例如 www.google.com) 翻譯成電腦看得懂的 IP 位址。

封包、路由、可靠度

當你傳送一張圖片時，網路會把它切成很多個小「封包」，每個封包都有目的地 IP 位址。

「路由器 (Router)」就像是網路上的交通警察，會指揮這些封包走最快的路徑（路由）到達目的地。

如果中間有封包遺失了，TCP/IP 這個通訊協定會確保對方沒收到，並要求重傳，這就是「可靠度」。

HTTP 和 HTML

HTTP (超文本傳輸協定) 是瀏覽器和網站伺服器溝通的「語言」。當你在瀏覽器輸入網址，就是用 HTTP 語言說：「嗨！請把這個網頁的資料給我！」

而伺服器回傳給你的，就是用 HTML (超文本標記語言) 寫成的網頁原始碼，瀏覽器再把它翻譯成我們看到的漂亮網頁。

網路探險家挑戰

問答題：

  1. 把好記的網址 (www.google.com) 翻譯成數字 IP 位址的系統，叫做什麼？ (提示：三個英文字母)
  2. 負責指揮數據封包走最佳路徑的網路設備是什麼？
  3. 我們現在正在看的這個網頁，主要是由哪一種語言寫成的？

解答： 1. DNS, 2. 路由器 (Router), 3. HTML。 你都答對了嗎？

關卡三

網路安全與數位公民

保護自己，尊重他人

網路世界充滿樂趣，但也潛藏著一些危險。身為一個負責任的探險家，我們必須學會如何保護自己和尊重他人。

加密與公開金鑰

當你輸入密碼或刷卡時，資料是怎麼被保護的？靠的就是「加密」！

想像你有一個只能用你的「私鑰」才能打開的專屬鎖頭（公開金鑰），你把這個鎖頭（公開金鑰）複製給很多人。別人要寄機密信給你時，就用你的鎖頭鎖上，這樣全世界只有擁有「私鑰」的你才能打開這封信，確保了資料安全。

網路安全與犯罪

網路犯罪有很多種，例如：

  • 釣魚網站：製作假的網站騙取你的帳號密碼。
  • 病毒/惡意軟體：破壞你的電腦或竊取資料。
  • 網路霸凌：在網路上用言語攻擊或散播不實言論。

記住，在網路上，我們也要遵守法律和道德，不做傷害別人的事。

關鍵字搜尋的原理

為什麼 Google 搜尋這麼快？因為它有許多「網路爬蟲」機器人，不斷地在網路上爬取、索引和儲存全世界的網頁資料。

當你輸入關鍵字，Google 不是當下才去找，而是從它龐大的「索引資料庫」中，瞬間找出最相關的結果給你。這就是為什麼搜尋幾乎是即時的！
