CSS 是什麼？

賦予網頁視覺風格的魔法

上次我們學到，HTML 是網頁的「骨架」。那麼 CSS (Cascading Style Sheets) 就是網頁的「造型師」。它專門負責打理網頁的外觀，決定了所有視覺呈現的細節。

透過 CSS，我們可以控制：

  • 顏色：文字顏色 (color)、背景顏色 (background-color)。
  • 字體：字型 (font-family)、大小 (font-size)、粗細 (font-weight)。
  • 空間：邊距 (margin)、內距 (padding)。
  • 排版：如何排列元素 (display: flex)、對齊方式等。

如果沒有 CSS，所有網頁都會是黑白文字加上藍色連結的樸素模樣。CSS 讓網頁設計充滿無限可能。
