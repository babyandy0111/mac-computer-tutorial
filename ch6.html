JavaScript 是什麼？

網頁的「大腦」與「肌肉」

我們已經有了骨架 (HTML) 和外觀 (CSS)，但網頁還只是個漂亮的展示品。JavaScript (簡稱 JS) 就像是網頁的「大腦與肌肉」，它負責處理所有的「互動行為」。

JS 是一種真正的程式語言，可以讓你的網頁回應使用者的操作：

  • 當使用者點擊按鈕時，執行特定任務 (例如送出表單)。
  • 改變網頁上的內容或 CSS 樣式，而不需要重新載入頁面。
  • 製作圖片輪播、下拉式選單等動態效果。
  • 驗證使用者在表單中輸入的資料是否正確。

簡單來說，如果你想讓網頁不只是「看」，還能「玩」和「用」，你就需要 JavaScript！
